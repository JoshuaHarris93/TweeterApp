# Tweeter Project

Tweeter is a simple, single-page Twitter clone. It uses a combination of Express, AJAX, CSS, HTML, jQuery, and MongoDB.

## Getting Started

Install dependencies using the `npm install` command.

## Dependencies

- MongoDB
- Express
- Node 5.10.x or above
- Body Parser
- Chance
- MD5
